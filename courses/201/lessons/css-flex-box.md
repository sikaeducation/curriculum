# CSS: Flexbox

## Lesson Objective

Layout elements with CSS Flexbox.

## Check For Understanding

Make this list display horizontally, evenly-spaced across the container.

```js
<nav>
  <ul>
    <li>
      <a href="#">A</a>
    </li>
    <li>
      <a href="#">B</a>
    </li>
    <li>
      <a href="#">C</a>
    </li>
  </ul>
</nav>
```
